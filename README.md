# Dashboard HV

Dashboard de análisis de ventas por supermercados.

## Cómo correrlo localmente

```bash
npm install
npm run dev
```

## Cómo publicar en Vercel

1. Sube este proyecto a GitHub (un repositorio nuevo)
2. Ve a [vercel.com](https://vercel.com) e inicia sesión con GitHub
3. Haz clic en **"Add New Project"** e importa el repositorio
4. Vercel detectará automáticamente que es un proyecto **Vite + React**
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. Haz clic en **Deploy** ✅

En 2-3 minutos tendrás una URL pública como:
`https://dashboard-hv-tuusuario.vercel.app`

## Tecnologías usadas

- React 18
- Vite
- Tailwind CSS
- Lucide React (íconos)
